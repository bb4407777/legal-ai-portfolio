## 项目：Lawlink 法律业务管理系统

*独立开发 · 技术栈：Next.js 16 / TypeScript / Prisma 5 / PostgreSQL / Tailwind CSS / Docker*

[github.com/bb4407777/LawLink](https://github.com/bb4407777/LawLink)

> 3 年持续迭代，189+ 次提交，管理 706 件真实案件、2,034 份文书、1,293 条办案记录

**法律数据模型设计：**
- 设计覆盖案件全生命周期的法律数据模型，包括案件实体、当事人关系、法院层级、程序节点、期限规则、文书结构化等核心 Schema
- 沉淀可复用的法律知识数据体系，将办案经验转化为结构化数据资产

**自动化工作流引擎：**
- 12368 短信自动解析 → 创建待办任务 → 同步系统日历 → 到期自动触发提醒
- 庭审信息提取后自动写入 Apple 日历，零人工干预
- 案件生命周期状态机驱动，关键节点自动推进

**OCR 证据管道：**
- 集成微信聊天记录截帧 OCR 管线
- 区分左右气泡（左=对方，右=我方），实现证据材料自动结构化
